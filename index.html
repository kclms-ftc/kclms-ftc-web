Think Award Winners

We won the Think Award at our first UK qualifier. The judges give it for the engineering work and documentation behind a robot rather than for match results. The portfolio they read is on this site.

The Robot

Mecanum drive, three-wheel odometry, Pedro Pathing and a Limelight vision system we integrated ourselves. We prototype in hardware and model the full machine in CAD at the same time. All of the code is public.

Technical Details

This Season: DECODE

Robots read a coded sequence with computer vision, then launch artifacts into goals in that order. Scoring rewards accuracy, so our design centres on a tuned dual-flywheel launcher and a repeatable autonomous routine.
