The Machine

Built in-house for the DECODE challenge, season 2025–26.

How We Build It

We design the robot in two ways at the same time. We build rough prototypes to test on the field, and we model the whole machine in CAD before cutting parts. Most subsystems went through a few versions before they worked. The drivetrain, launcher and vision stack were all built and tuned by the team, and the code is in our open-source repository.
